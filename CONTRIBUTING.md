# Přispívání do projektu

Děkujeme za váš zájem o přispívání do PDF Extractor projektu!

## Jak přispět

1. **Fork** repozitář
2. Vytvořte **feature branch** (`git checkout -b feature/AmazingFeature`)
3. **Commit** vaše změny (`git commit -m 'Add some AmazingFeature'`)
4. **Push** do branch (`git push origin feature/AmazingFeature`)
5. Otevřete **Pull Request**

## Pravidla pro commit zprávy

- Používejte popisné commit zprávy
- Začněte s velkým písmenem
- Používejte imperativní tón ("Add feature" místo "Added feature")

## Kódování

- Dodržujte PEP 8 pro Python kód
- Používejte type hints kde je to možné
- Přidávejte docstringy k funkcím a třídám

## Testování

Před odesláním PR prosím:
- Otestujte své změny na Windows
- Ověřte, že všechny existující funkce stále fungují
- Zkontrolujte, že nejsou žádné syntax errors

## Dokumentace

- Aktualizujte dokumentaci pokud je to nutné
- Přidejte komentáře k složitějším částem kódu
- Aktualizujte CHANGELOG.md pokud přidáváte nové funkce

## Otázky?

Pokud máte jakékoliv otázky, otevřete issue nebo kontaktujte maintainery projektu.
